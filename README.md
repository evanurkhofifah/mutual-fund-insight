# Mutual Fund Fact Sheet Analyzer

Aplikasi **Streamlit** interaktif yang menggunakan **AI (LLM)** untuk menganalisis **Fund Fact Sheet** (reksa dana) dan memberikan ringkasan informasi penting serta **Risk Insight** sesuai profil risiko user. Aplikasi ini berbasis Python dan LangChain + Google Gemini AI.

---

## Fitur Utama

- Upload **PDF Fund Fact Sheet**.  
- Pilih **profil risiko** user: Conservative, Moderate, Aggressive.  
- Klik **Start Chat** untuk memulai interaksi dengan chatbot.  
- Chatbot memberikan ringkasan fund fact sheet termasuk:
  - Jenis reksa dana
  - Alokasi aset
  - Biaya / fee
  - Nilai AUM
  - Return historis (1Y, 3Y, YTD)
  - Risk Insight sesuai profil risiko user
- Interaksi chatbot menggunakan **Natural Language Processing (NLP/LLM)**, sehingga user dapat bertanya secara bebas.
